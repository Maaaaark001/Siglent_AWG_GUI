## Siglent_AWG_GUI
鼎阳AWG的开源GUI上位机，由PYQT5设计，Visa通信，意在取代计算很慢的EasyWave

## 必要条件
1. Python<=3.8
2. visa 驱动
3. pyqt5
4. pyvisa
5. numpy
6. matplotlib

## 文件结构树
```
Siglent_AWG_GUI
│  GUI.py --GUI主程序
│  GUI.ui --GUI界面文件
│  LICENSE --许可文件
│  mplwidget.py --matplotlib绘图控件
│  README.md --说明文件
│  run.bat --运行命令
│  sdg_wavedata_demo.py --鼎阳py 官方demo
```
